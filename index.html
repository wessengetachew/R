Overview

This explorer provides an interactive research environment for investigating an unsolved problem in number theory posed by Wacław Sierpiński in 1964. The tool enables visualization and analysis of positive integers that cannot be expressed in any of the four forms involving the expression 6ab ± a ± b where a and b are positive integers.

Research Question

Are there infinitely many positive integers that cannot be expressed as any of:

  • (3a + 1)(2b + 1) - 1
  • (3a + 1)(2b - 1) - 1
  • (3a - 1)(2b + 1) - 1
  • (3a - 1)(2b - 1) - 1

for positive integers a and b?

Mathematical Background

The Sierpiński Problem

In 1964, Wacław Sierpiński investigated the representation of positive integers through specific bilinear forms. The problem concerns whether infinitely many positive integers remain uncovered by the four forms:

Form 1: n = 6ab + a + b = (3a + 1)(2b + 1) - 1 Form 2: n = 6ab + a - b = (3a + 1)(2b - 1) - 1 Form 3: n = 6ab - a + b = (3a - 1)(2b + 1) - 1 Form 4: n = 6ab - a - b = (3a - 1)(2b - 1) - 1

This problem remains open and represents a fundamental question about the structure of positive integers.

Mathematical Connections

Prime Number Theory

All primes p > 3 satisfy p ≡ ±1 (mod 6). This modular constraint connects directly to the structure of the four Sierpiński forms, each involving coefficients related to this fundamental property.

GCD Structure

The coverage patterns exhibit enhanced structure when GCD(a,b) divides 6. This observation suggests deep connections to divisibility properties and modular arithmetic.

Search Optimization

For n = 6ab ± a ± b with a ≤ b, mathematical analysis shows that a ≤ √(n/6).

This bound significantly reduces the computational complexity of determining coverage.
